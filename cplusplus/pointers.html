Pointers

A pointer in c++ is a variable that stores the address of another variable. An address is a number that represents a particular location in memory (RAM).

When a pointer variable is created, the type of the pointer indicates the type of data stored at the memory location pointed to by the pointer. For example:

int* x;

declares a variable that can hold a pointer to an integer. Note that in this example, the value (address) of the pointer is not specified, which is generally a bad idea. The best practice is to always initialize variables as soon as you create them.

It is important to remember that the value of a pointer is an address. The variable x declared above does not store an integer, it stores the address of an integer.
